Wine Quality Prediction API Documentation

This documentation provides detailed information about using the Wine Quality Prediction API.

Overview

The Wine Quality Prediction API allows for the prediction of wine quality based on chemical properties. This API uses a trained machine learning model and accepts specific attributes of wine for analysis.

API Endpoint

POST /predict

This endpoint accepts JSON data representing the attributes of a wine sample and returns a prediction of its quality.

Request Format

The request must be in JSON format with the following attributes:

  • fixed acidity: Float
  • volatile acidity: Float
  • citric acid: Float

Response Format

The response will be a JSON object containing the predicted quality rating:

{"predicted_quality": 6}

Error Handling

The API provides descriptive error messages in case of invalid input. For example:

{"error": "Invalid input for 'fixed acidity'. Expected a numeric value."}

Example Request

An example of a valid POST request to the /predict endpoint:

fetch('/predict', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    "fixed acidity": 7.4,
    ...
    "type": 1
  })
});

Example Response

An example response from the API might look like:

{"predicted_quality": 6}
